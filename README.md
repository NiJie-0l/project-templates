# project-templates

Starter templates for new projects. Use with `gh repo create --template`.

## Templates

| Template | Description |
|----------|-------------|
| `python-app` | Python application with pytest, CI, MIT license |
| `node-app` | Node.js app with ESLint, CI, MIT license |
| `static-site` | Static site with GitHub Pages deployment |

## Quick start

```bash
# Create from template
gh repo create my-project --template NiJie-0l/project-templates

# Or use workspace init script
~/Desktop/workspace/core/scripts/workspace-init.sh my-project public python
```
